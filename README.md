# Beta Engine
A lightweight component baised game engine that is built on GLFW.

# Beta Level Editor
A Level Editor for the Beta Engine. It can be used to make tilemaps and place gameobjects around diffrent levels.

## Features
- Edit/Create Tilemaps
- Place Gameobjects
